# Personal Firewall Using Python

## Objective
To develop a lightweight personal firewall that filters network traffic based on predefined rules.

## Features
- Packet sniffing using Scapy
- Rule-based traffic filtering
- Website blocking using iptables
- Firewall activity logging
- GUI for monitoring

## Tools Used
- Kali Linux (Virtual Machine)
- Python 3
- Scapy
- iptables
- Tkinter
